## Phase 3: What I Built

### Interactive ROI Calculator — `/tools/roi`

**New files:**
- `app/tools/roi/page.tsx` — Server component with full SEO metadata, methodology section
- `components/roi/ROICalculator.tsx` — Client component with 4 reactive sliders
- `lib/copy/roi.ts` — All copy strings (single source of truth, matching existing pattern)
- `app/globals.css` — Custom slider CSS (`.roi-slider` with `--slider-pct` CSS custom property for fill)

**Features:**
- **4 input sliders:** Artists in roster (1-500), hours/week per artist (1-20), team hourly cost ($20-200), posts/week per artist (1-20)
- **Animated result cards:** Numbers count up with ease-out cubic animation on every change
  - Annual time saved (hours)
  - Annual cost saved ($) — highlighted card
  - Content output increase (%)
  - ROI multiple (N×) — highlighted card
- **Comparison bar:** Visual side-by-side of weekly hours "Without Recoup" vs "With Recoup"
- **Dual CTAs:** "Start free" → app, "Book a demo" → /advisory/book
- **Methodology section:** Transparent about assumptions (80% automation, 2.5× content amplification)
- **Responsive:** Works on mobile with stacked layout

**Calculation logic:**
- 80% automation rate (conservative)
- 2.5× content amplification with AI
- Pricing tiers: $19/mo (solo), $99/mo (≤10 artists), $15/artist/mo (11+)
- ROI multiple = net savings / Recoup cost
