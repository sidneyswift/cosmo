## Phase 3: What I Built

### Interactive Artist Intelligence Demo

**What:** A new page at `/learn/demos/artist-intelligence` where visitors type any artist name and see live Recoup intelligence data.

**Files created:**
1. `app/api/demo/research/route.ts` — API proxy route
   - Proxies to Recoup Research API with built-in demo key
   - Tries Chartmetric-backed endpoints (profile, cities, similar, insights) first
   - Falls back to web intelligence (`POST /research/web`) if Chartmetric is unavailable
   - Rate limited: 1 request per 5 seconds per IP (in-memory, auto-clears)
   - Graceful error handling — returns partial data, never crashes

2. `app/learn/demos/artist-intelligence/page.tsx` — Page wrapper with SEO metadata

3. `app/learn/demos/artist-intelligence/ArtistIntelligenceDemo.tsx` — Interactive client component
   - Search input with quick-select buttons (Joy Crookes, Drake, Billie Eilish, Tyler the Creator)
   - **Full results mode** (when Chartmetric works): streaming snapshot (Spotify/IG/TikTok/YT), playlist position, top listener cities, competitive landscape, AI insights
   - **Web fallback mode** (current): web intelligence results with upsell messaging explaining the full plugin provides structured analytics
   - Loading skeleton animation, error states, responsive design
   - CTA at bottom: "Want this for your entire roster?" → /pricing
   - Matches existing Recoup dark mode design system exactly

4. Updated `app/learn/demos/page.tsx` — replaced placeholder with demo listing showing "Artist Intelligence" with "Live" badge

### Discovery: Chartmetric API Auth Issue

While building, discovered the Recoup API's Chartmetric backend is returning 401 errors on all CM-backed endpoints (`/profile`, `/cities`, `/similar`, `/insights`). The web intelligence endpoints (POST) work fine. This is a backend issue — the CM token needs rotation.

**Workaround built:** The demo automatically falls back to web intelligence, which returns rich data from ChartMasters, Music Metrics Vault, and other sources. When Chartmetric auth is fixed, the demo will automatically upgrade to structured analytics with no code changes.

**Bug filed mentally:** Sid or Sweets should check the Chartmetric token rotation in the API backend. All free-tier API keys get 401 on CM endpoints.
