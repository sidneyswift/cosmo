## Phase 3: What I Built

### `recoup-getting-started` skill

Full onboarding flow for new agents:
1. **Validates credentials** — tests API key against `/accounts/me`, reports account info
2. **Discovers capabilities** — shows all 4 Recoup plugins with install commands, recommends based on use case (managers → research+content, investors → catalog diligence)
3. **Runs a demo query** — artist search to prove the pipeline works end-to-end
4. **Recommends next steps** — one concrete action based on installed plugins

Decision tree routes agents to the right step based on their question ("is my key working?" vs "what can I do?" vs "show me it works").

### `recoup-health-check` skill

Diagnostic tool for troubleshooting:
1. API connectivity (no-auth test)
2. Authentication validation
3. Credit balance check
4. Plugin inventory (scans for known skill directories)
5. Functional test (lightweight search query)
6. Common issues reference table with specific fixes
