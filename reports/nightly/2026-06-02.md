## Phase 3: What I Built

### `llms.txt` — Concise Agent Entry Point (50 lines)

The llms.txt standard (llmstxt.org) is what AI agents look for when they encounter a new API. It's the equivalent of robots.txt for AI.

Contents:
- What Recoup is (AI infrastructure for music business, BYOA strategy)
- Quick start: base URL, auth, instant API key creation
- All 4 plugins with install commands and descriptions
- 10 API groups with 1-line descriptions
- Links to full docs and expanded reference

### `llms-full.txt` — Complete Agent Reference (534 lines)

Everything an agent needs to use Recoup without reading HTML docs:
- Authentication walkthrough (programmatic and manual)
- All 4 plugins with their commands and skills listed
- 4 workflow recipes: research an artist, create content, analyze a catalog deal, set up automation
- All 161 API endpoints organized by group with descriptions
- Extracted automatically from the 9 OpenAPI spec files
