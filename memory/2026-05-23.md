## Content Engine Simplification Round 2: Single Database (12:30 AM - 1:30 AM)

### What Changed
Killed the SQLite↔Supabase sync and made **Supabase the single source of truth** for all draft operations. This was the #2 item on the simplification list from the "10x engineer" diagnosis.

### Before
- Two databases: SQLite (`data/content.db`) and Supabase (`app_content_dashboard` schema)
- `sync-to-supabase.js` pushed local → Supabase every 2 minutes
- `sync-from-supabase.js` pulled Supabase → local every 2 minutes
- Constant drift: status conflicts, race conditions, Sid's dashboard feedback delayed 12+ hours before the fix on 5/22
- Every script imported `db.js` directly and did raw SQLite queries for drafts

### After
- **Supabase = single source of truth for drafts** (reads + writes)
- **SQLite retained for**: ideas, performance, voice_profile, topic_multipliers (local-only, no dashboard)
- **`db-adapter.js`** rewritten as comprehensive adapter:
  - Draft operations → Supabase (getDrafts, insertDraft, updateDraft, deleteDrafts, countDraftsByStatus, etc.)
  - Idea/performance operations → SQLite (unchanged)
- **13 scripts migrated** to use dbAdapter for draft operations:
  - generate.js, brain-first-generate.js, regenerate.js, qa-judge.js, approve.js
  - auto-slides.js, nightly-engine.js, pipeline-loop.js
  - daily-digest.js, morning-report.js, perf-check.js, self-improve.js
- **3 scripts archived** (no longer needed):
  - `sync-to-supabase.js` → `_archive/`
  - `sync-from-supabase.js` → `_archive/`
  - `slack-bridge.js` → `_archive/` (legacy Slack approval workflow, replaced by dashboard)
- **Pipeline loop updated**: 2-min cycle is now `regenerate → qa-judge → auto-slides` (no sync step)
- **Supabase migration applied**: Added missing columns (postiz_post_id, slack_message_ts, slack_channel, last_reaction_check, parent_draft_id) + indexes

### Migration SQL
`supabase/migration-002-single-db.sql` — run against `$SUPABASE_DB_URL`

### Architecture Now
```
Supabase (app_content_dashboard)
├── drafts    ← single source of truth (all scripts read/write here)
├── ideas     ← exists but not actively used by new scripts
└── activity_log

SQLite (data/content.db)  
├── ideas          ← discovery, triage, POV generation
├── performance    ← analytics cache
├── voice_profile  ← voice calibration
└── topic_multipliers ← feedback loop scoring
```
