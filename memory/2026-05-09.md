## #cosmo-review System Built (07:52)

### Problem
Sid couldn't review my work — had to scroll through messages, no way to see changes (especially marketing site) from his phone, no approval gate before things go live.

### Solution: Review Queue in #cosmo-review (C0B2PFHTP2A)
- Created `REVIEW.md` — full spec for the staging/approval workflow
- Every external-facing artifact gets a review card posted to #cosmo-review before going live
- Review cards include: description, context, preview (Vercel preview URL / inline content), and approval instructions
- Reactions: 👍 approve, 💬 thread reply for feedback, 👎 reject, ⏸️ hold
- No localhost URLs — everything must be phone-reviewable
- I block on pending reviews (don't stack work in same category)
- Updated AGENTS.md, TOOLS.md, HEARTBEAT.md to integrate the new flow
- Heartbeats now check #cosmo-review for reactions every cycle

### Key Rule: Nothing goes live without approval
- Code: push to branch → Vercel preview → review card → approval → merge
- Content: full copy + images in review card → approval → publish
- Config/infra: what changed + why → approval → apply
